# Student Management System (Python)

## Description
This is a simple Student Management System developed using Python.  
The program allows users to manage student records through a command-line interface.

## Features
- Add a new student
- View all students
- Search student by roll number
- Delete student record
- Menu-driven program

## Technologies Used
- Python
- Dictionary data structure
- Functions
- Loop and conditional statements

## How It Works
The program stores student data (roll number, name, and marks) using a Python dictionary.  
Users interact with the system through a menu where they can perform different operations like adding, viewing, searching, and deleting student records.
